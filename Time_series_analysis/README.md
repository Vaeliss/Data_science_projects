Project for the business, economic and financial data exam (21/22) in unipd.

The focus of the project is to investigate past trends in the music industry, current financial status of Spotify and what drives popularity of the songs on Spotify.

Three datasets are used. One from RIAA for historical data, one from spotify earning reports and one from spotify API.

The project is carried out using R programming language. DIMORA package is necessary to run it.
